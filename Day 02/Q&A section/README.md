# ☕ Java Q&A Session

This README contains three basic Java practice questions.

---

## ✅ Question 1

**Problem:**
Get input for variables **Name** and **Age**. Print it.

### 📝 Sample Input

John
24

### 🖥 Sample Output

My name is John
My age is 24

---

## ✅ Question 2

**Problem:**
Get input for variables **Name, Age, and Address**. Print it.

### 📝 Sample Input

John
24
Chennai

### 🖥 Sample Output

My name is John
My age is 24
My address is Chennai

---

## ✅ Question 3

**Problem:**
Get input for 3 integer variables: **a, b, and c**.
Multiply all variables and store it in **d**.
Add all variables and store it in **e**.
Divide **d by e** and print the result.

### 📝 Sample Input

10
20
10

### 🖥 Sample Output

50

---

💡 These questions are designed to practice basic input, output, arithmetic operations, and variable handling in Java.
