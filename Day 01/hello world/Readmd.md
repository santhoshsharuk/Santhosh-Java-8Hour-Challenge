---

## 📝 Hello World Program

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

---

## 🔍 Code Explanation

### 1️⃣ public class Main

* Every Java program must have a class.
* The file name must match the class name (Main.java).

### 2️⃣ public static void main(String[] args)

* This is the entry point of the program.
* JVM starts execution from this method.

### 3️⃣ System.out.println()

* Used to print output to the console.
* println() prints and moves to the next line.

---

## ▶️ How to Compile and Run

### Step 1: Check Java Version

```
java -version
javac -version
```

### Step 2: Compile the Program

```
javac Main.java
```

### Step 3: Run the Program

```
java Main
```

---

## ✅ Output

```
Hello, World!
```
